About KTeach

The Kotlin Classroom that's easy to understand.

Learn Kotlin Confidently

KTeach was a Final Project idea for CS 199-KT, a Kotlin class at UIUC. It was built to give professors a platform to create their own custom Kotlin curriculums with ease.

Simplicity is Everything

Kotlin is a challenging language to learn, especially if you're not used to topics such as null safety or concurrent programming. The UI was designed to be inuitive, such that professors just have to worry about creating problems and students just have to focus on learning Kotlin.
